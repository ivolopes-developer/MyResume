Nasty Boy's - Metin2 Guild

Project information

  • Category: Web / Desktop
  • Client: Personal Project
  • URL: nastyboys.netlify.app

I was doing a great job playing Metin2, when I heard a member of my guild say "It's annoying that we have to be online to recruit people".

It crossed my mind to create a website for the guild whose objective is to allow players to get to know the guild and be able to apply through the website, which in turn will be sent to discord, notifying the administrators. This is done through webhooks.


  • Development Environment
  • ReactJS
  • Typescript
  • Netlify (to host react website)
  • AntDesign
