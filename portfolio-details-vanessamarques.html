Project information

  • Category: Web Development
  • Client: Vanessa Marques
  • URL: www.vanessamarques.pt

Vanessa Marques is an architecture and interior design firm. This project aims to be the brand's web space where it will be known to its target audience.

The development of this project was in partnership with a brand designer who provided all the necessary material to the web designer so that the website design was consistent with the Vanessa Marques brand. Later it facilitated my development work because the design plan was already 100% stipulated.

To facilitate user interaction with the website, I created a system within the WordPress dashboard with pages where Vanessa can edit the website's content. You can edit texts on the home page, you can insert your own projects without needing to ask a developer.

If she later wants to receive the contacts in another email, she can also easily edit the email to which the responses to the form are sent.
