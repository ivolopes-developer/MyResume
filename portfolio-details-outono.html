Outono

Project information

  • Category: Desktop App
  • Client: Personal Project

"Outono" is a management software made with small and medium-sized companies in mind. This project was developed using Java and SQL language. The software works like this: there are common user and administrator accounts, everything can be managed within the software. There are several features such as: view registered products, register an invoice, insert stock, consult invoices, print documents, among others...

"The Shed" was a startup I created with my friend Diogo Francisco to participate in a contest organized by CDI Portugal called: "App's for Good".

note: This application is in Portuguese only.
