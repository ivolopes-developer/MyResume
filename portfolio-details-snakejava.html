Snake Java

Project information

  • Category: Desktop Game
  • Client: Personal Project

In the first year of my course, I had to do a final year project in C. As I like video games and in my group of friends we like to use the "Nokia 3310" meme, I decided to make the Snake game on a console.

So in the second year of the course, when I started learning JAVA I decided to make a remake of Snake, with various levels of difficulty and already with an user interface.

So... the game works, and there were classmates who liked it, so I created an .exe file for the game and used winRAR to create an installer. This way it is not necessary to transfer the code to each PC.
